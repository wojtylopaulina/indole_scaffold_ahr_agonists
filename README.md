# QSAR Model for Indole-Scaffold AhR Agonists

This repository contains a machine learning-based QSAR model developed for the prediction of Aryl Hydrocarbon Receptor (AhR) agonists within indole-scaffold compounds.

## Author
Paulina Anna Wojtyło  
ORCID: https://orcid.org/0000-0003-2740-776X

## Description
The model was developed using MLJAR AutoML and is intended for research applications in cheminformatics and drug discovery.
  
## Data Sources
The model was trained on curated datasets derived from publicly available sources, including databases, literature, and patents.
